Architecture

ML pipeline

Market Data → Feature Engineering → News Sentiment → XGBoost Model → Meta-labeling → Signal Generator → Backtest

Features used

  • Multi-horizon returns (1d to 60d)
  • Technical indicators (RSI, MA crossovers, volatility)
  • Volume dynamics and momentum
  • Cross-asset signals (SPY, QQQ, VIX)
  • Market regime features
  • News sentiment (historical + Finnhub API)

Model details

  • Separate XGBoost model per stock
  • Meta-labeling via Logistic Regression
  • Walk-forward validation (rolling splits)
  • Hyperparameter tuning with grid search
  • Threshold optimisation per confidence level
  • Daily automated retraining on Databricks
